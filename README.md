# Website LPM MTS DARUL ULUM

Website resmi LPM MTS DARUL ULUM yang dibangun dengan HTML5, CSS3, dan JavaScript vanilla. Website ini menampilkan profil sekolah, berita, galeri, ekstrakurikuler, guru & staf, dan informasi kontak dengan desain gradasi hijau-putih yang elegan dan modern.

## Fitur

- Desain modern dengan gradasi hijau-putih yang elegan
- Slider otomatis di halaman utama
- Sambutan kepala sekolah dengan tanda tangan digital
- Berita terbaru dengan fitur pencarian
- Ekstrakurikuler unggulan dengan detail modal
- Galeri foto dengan lightbox dan filter kategori
- Profil guru & staf dengan filter jabatan
- Form kontak dengan validasi
- Peta lokasi sekolah
- Animasi scroll dengan AOS
- Tombol kembali ke atas
- Header yang berubah saat di-scroll
- Preloader animasi
- Responsif untuk semua perangkat
